# ST Vision

A SillyTavern extension for AI image generation and management using NovelAI.

## 1. Installation

See [here](https://docs.sillytavern.app/extensions/#third-party-extensions)

## 2. Development

```bash
# compile the extension
npm run build

# format code
npm run format
```
